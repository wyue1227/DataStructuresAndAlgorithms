### 面试题
如果一颗完全二叉树有768个节点，求叶子结点个数
##### 答案
- n1: 度为1结点个数
- n2: 度为2结点个数
- n0: 为叶子结点
- n: 结点总数
- m: 分支总数
完全二叉树n1个数只能为0或1
```text
基本公式
    1. n=n0+n1+n2
    2. m=n1+2*n2
    3. n-1=m
1, 2, 3代入可得4
    4. n0=n2+1
1, 4代入可得5
    5. n=n1+2n2+1
代入768和5可得
    768=n1+2n2+1
因为768是偶数，所以n1只能为1，则n2=383
所以叶子结点n0=768-383-1=384
```
